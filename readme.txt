How to install the plugin:

	On the dashboard click onto the plugins options located on the left hand side of the panel.
	Choose the file called "backpacker's blog widget" and click activate. 

	A message will appear at the top confirming if that it has been activated successfully.

	Then you will be able add custom post types by country and it will show in a separate tab on your menu folder

Enjoy! :)
